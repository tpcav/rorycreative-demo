# rorycreative-demo

This repository an interactive website demo for Rory Creative

---

## Features
1. Coverflow-like ablum image art effect
2. Click on images for link to learn more on Instagram
3. Scroll with:
  - touch
  - keyboard (left & right arrows)
  - trackpad
  - mouse scroll wheel
